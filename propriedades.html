Propriedades

Propriedades e atributos são as mesmas coisas.



É tudo aquilo que a classe possui.

Exemplo:
Conta bancária

  • Banco
  • Nome do titular
  • Número da agência
  • Número da conta
  • Saldo

//definindo classes em php
class contaBancaria
{
$banco;
$nomeTitular;
$numeroAgencia;
$numeroConta;
$saldo;

}


$conta = new contaBancaria(); // criei uma instancia da classe (objeto)

Desse modo o PHP retornará erro, pois as propriedades precisam de modificadores de acesso.
