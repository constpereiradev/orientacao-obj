Visibilidade

Modificadores de acesso: define qual é a visibilidade dos elementos da class para elementos externos da class.

Tipos:

  • Publico (public)

    • Deixa o elemento público.
    Qualquer elemento de fora da classe consegue acessar os atributos.


  • Privado (private)

    • Deixa o elemento privado.
    Não é possível acessar os atributos de fora da class.


  • Protegido (protected)

    • Semelhante ao private.
    Bloqueia o acesso para elementos de fora da class. A única diferença é que possui uma excessão: Consigo acessar elementos protecteds em classes que herdam a class pai.


//definindo classes em php
class contaBancaria
{
public $banco;
public $nomeTitular;
public $numeroAgencia;
public $numeroConta;
public $saldo;

}


$conta = new contaBancaria(); // criei uma instancia da classe (objeto)
